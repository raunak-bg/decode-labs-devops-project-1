# decode-labs-devops-project-1

## Objective
Learn Linux and Command Line Basics.

## Commands Executed

```bash
mkdir -p app/logs
touch app/config.conf
echo "Started" > app/logs/server.log
pwd
ls -R app
mv app/logs/server.log app/logs/server.bak
ls -l app/config.conf
```

## Expected Final Structure

app/
├── config.conf
└── logs/
    └── server.bak
